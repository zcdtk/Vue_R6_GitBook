## 项目结构


可变字段说明（均在 iBizSys6 平台进行配置）：

- ***<部件标识>***: IBizSys 模型预置，如 `GRID`、`FORM` 等。
- ***<视图引擎代码名称>***: IBizSys 模型预置，如 `EditView`、`GridView` 等。
- ***<应用代码名称>***: 前端应用------代码名称。
- ***<应用模块代码名称>***: 前端应用------应用模块------代码名称。
- ***<实体代码名称>***: 数据模型------实体------代码名称。
- ***<视图代码名称>***: 前端应用------全部应用视图------视图------代码名称。
- ***<部件代码名称>***: 数据模型------实体------部件-------代码名称。
- ***<组件代码名称>***: 基础文件预置与用户自定义。
